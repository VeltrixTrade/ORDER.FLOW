# 🥇 XAU/USD Order Flow AI Trading Bot | بوت الذهب المطور بتدفق السيولة والذكاء الاصطناعي

بوت تلجرام احترافي ومستشار مالي مدعوم بالذكاء الاصطناعي **DeepSeek**، يختص في تحليل زوج الذهب/الدولار (XAU/USD) بالاعتماد على بيانات تدفق السيولة الحقيقية (Order Flow) من منصة ميتاتريدر 5 مباشرة، واستراتيجيات الحجم والسيولة العميقة.

Bilingual Telegram Bot (Arabic + English) integrating **DeepSeek AI** and mathematical Order Flow calculations to analyze the Gold market (XAU/USD) using real-time MT5 tick volume data.

---

## 🌟 المميزات | Features

1. **📊 تحليل تدفق السيولة الفوري (Real-Time Order Flow Engine)**:
   - استقبال تيكات الأحجام المباشرة من ميتاتريدر 5 وحساب تراكم أحجام الشراء والبيع (Cumulative Volume Delta - CVD).
   - رسم مخطط الحجم الحجمي الفوري (Volume Profile) وتحديد مستويات التحكم السعري (Point of Control - POC).
   - حساب حدود منطقة القيمة (Value Area High - VAH / Value Area Low - VAL) لحظياً.
   - رصد اختلالات أحجام الطلب والعرض قُطرياً (Diagonal Imbalances) والامتصاص الذمي في ذيول الشموع (Wick Absorption).

2. **📰 فلتر مفكرة الأخبار الاقتصادية التلقائي (Economic Calendar Filter)**:
   - يتصل البوت تلقائياً بمفكرة الأخبار لمراقبة الأحداث ذات التأثير العالي على الدولار (High Impact USD).
   - يتم تفعيل "تجميد الصفقات" تلقائياً قبل وبعد صدور الخبر بـ 15 دقيقة لحماية الحساب من الانزلاقات السعرية.

3. **💰 إدارة مخاطر صارمة وتأمين تلقائي (Risk Management & Autopilot)**:
   - لا يقدم البوت صفقة إلا بمعدل عائد إلى مخاطرة لا يقل عن 1:3.
   - تحديد نقطة الدخول (Entry)، وقف الخسارة (Stop Loss)، والهدفين (TP1, TP2).
   - **تأمين تلقائي للربح**: بمجرد تحقيق السعر للهدف الأول TP1، يقوم البوت تلقائياً بنقل وقف الخسارة إلى نقطة الدخول (Break-Even) ويبث إشعاراً فورياً للمشتركين.

4. **💬 محادثة حرة مع خبير الذهب الذكي**:
   - إمكانية فتح محادثة تفاعلية مباشرة مع المساعد الذكي لمناقشة سلوك الأسعار وحركة الذهب.

## ⚙️ التثبيت والتشغيل | Installation & Setup

### 1. تثبيت المكتبات | Install Requirements
افتح محطة الأوامر (Terminal/CMD) في مجلد المشروع ونفذ الأمر التالي لتثبيت المكتبات اللازمة:
```bash
pip install -r requirements.txt
```

### 2. ضبط المتغيرات البيئية | Configure API Keys
قم بتعبئة بياناتك السرية بداخل ملف `.env` في المجلد الرئيسي:
```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
TELEGRAM_CHANNEL_ID=-100xxxxxxxxxx
DEEPSEEK_API_KEY=your_deepseek_api_key_here
DEEPSEEK_MODEL=deepseek-chat
MT5_SECRET_KEY=NEROFLOW_SECRET        # المفتاح السري المعتمد لمصادقة ميتاتريدر
```

### 3. تشغيل البوت والسيرفر البوابة | Run the System
لتشغيل البوت وسيرفر استقبال الأسعار معاً، نفذ الأمر التالي:
```bash
python bot.py
```

### 4. ربط منصة ميتاتريدر 5 | MT5 Integration
1. قم بنسخ ملف **`NeroFlowStreamer.mq5`** وضعه بداخل مجلد إكسبرتات/مؤشرات منصة ميتاتريدر 5 الخاص بك (`MQL5/Indicators`).
2. افتح شارت الذهب بفريم 5 دقائق (XAUUSD, M5) واسحب المؤشر للشارت.
3. تأكد من تفعيل خيار **`Allow WebRequest`** في إعدادات المنصة وإضافة الرابط الخاص بك.
